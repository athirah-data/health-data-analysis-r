# health-data-analysis-r

This repository demonstrates basic health data analysis workflows using R, including data cleaning, descriptive statistics, reliability analysis, and logistic regression.

Tools used:
- R
- tidyverse
- dplyr
- ggplot2

Note: All data used are simulated or anonymized for learning and demonstration purposes. Analyses demonstrated include reliability testing, prevalence estimation, chi-square tests, and logistic regression.
